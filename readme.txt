Allows you to easily assign bbPress topics to users, as well as the ability to mark topics resolved. Also provides an admin page for listing topics by assigned user.

== Installation ==

= Manual Installation =

1. Upload the entire `/hwp-bbpress-topic-assignment` directory to the `/wp-content/plugins/` directory.
2. Activate HWP bbPress Topic Assignment through the 'Plugins' menu in WordPress.
